入门

  • 我的编程历史是入手golang语言的，golang语言的前身是c语言的作者，是google一群人开发的语言，所以也可以导入c语言的包来使用，可以说非常方便了，作为入门编程语言来说还是非常的nice。

  • go语言的语法↓

  • package mian
    import “fmt”
    func main(){
    fmt.Println(“Hello World!”)
    }

  • 是的，你没有看错，这就是一个Hello World的程序，语句结束后不用加分号，不和c/c++和java一样语句后面要加分号，在go语言中分号表示换行。而且并发是go语言的特点，高效并发使得go语言名声大燥，但是热门程度还是没有c/c++、Java和python热门
